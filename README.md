# Travelog

A full-stack travel journaling app built with React, Tailwind CSS, and Express.

<img width="1653" height="810" alt="Travelog screenshot" src="https://github.com/user-attachments/assets/1fb3d4b6-7b6b-4098-b713-92c1a6756f84" />

## Tech Stack

**Client**
- React 19
- Vite
- Tailwind CSS v4
- Lucide React (icons)

**Server**
- Node.js
- Express 5
- dotenv, cors

## Features

- Responsive navigation bar
- Tailwind-powered global styling
- Client/server monorepo with a single command to run both

## Getting Started

### Prerequisites
- Node.js (v18+)

### Installation

```bash
git clone https://github.com/Wuuuuu5/Travelog2.0.git
cd Travelog2.0
npm run install:all
```

### Running the app

```bash
npm run dev
```

This starts the client (Vite dev server) and server (Express + nodemon) concurrently.

## Project Structure

```
Travelog2.0/
├── client/     # React + Vite frontend
```
